Ласкаво просимо до EdduBuddy

Проект EdduBuddy створено для автоматизації навчальних процесів через інтерактивного чат-бота. Цей інструмент допомагає студентам задавати питання та отримувати відповіді на основі заданих матеріалів.

Команди

  • start - Почати нове навчання з ботом.
  • ask [питання] - Задати питання чат-боту.
  • material [тема] - Отримати матеріали по певній темі.
  • exit - Завершити сесію з чат-ботом.

Структура проекту

eddubuddy.py    # Основний скрипт для запуску чат-бота.
docs/
    index.md  # Головна сторінка документації.
    materials/ # Каталог з навчальними матеріалами.
    bot/       # Файли для роботи чат-бота.

Як користуватися

Щоб почати використовувати EdduBuddy, спочатку потрібно запустити основний скрипт eddubuddy.py на вашому сервері. Після цього бот стане доступним для спілкування.

Налаштування проекту

Щоб налаштувати проект, вам потрібно встановити необхідні бібліотеки за допомогою pip:

pip install -r requirements.txt

Після цього ви зможете запустити бота за допомогою команди:

python eddubuddy.py
